<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → updatePlanPrices action | Client submits { tierId, newMonthlyFeeCents } per tier. Server must validate ownership + that plan is in published state + that new prices are sane. |
| Server action → Stripe | Creating prices is mutating; retries must not create duplicates. |
| DB writes across tiers | Multi-tier price edit must be atomic: either all tiers get new prices or none do. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-04-01 | Tampering | Client sends newMonthlyFeeCents = 1 (penny plan) | mitigate | Zod validation server-side: newMonthlyFeeCents ∈ [500, 100_000] (i.e., $5 to $1,000). Outside range → VALIDATION_FAILED. |
| T-03-04-02 | Tampering | Client sends tierId belonging to another clinic | mitigate | withClinic(clinic.id, ...) — tier lookup returns null for foreign tierId. |
| T-03-04-03 | Repudiation | Owner disputes when a price change happened | mitigate | stripePriceHistory is append-only; every edit adds {priceId, unitAmountCents, createdAt, replacedAt}. The old entry's replacedAt is stamped at edit time. |
| T-03-04-04 | Denial of Service | Retry storm → N duplicate Stripe Prices | mitigate | Idempotency key: `price-edit:{planId}:{tierId}:v{N}:{unitAmountCents}` where N is the next version number (count of history entries + 1). A retry replays the same key → same price_id. |
| T-03-04-05 | Tampering / Business | Owner lowers price, existing subs now pay MORE than new enrollments advertised | mitigate via UX | Dialog discloses: "Applies to new enrollments only. Existing members keep their current price." The mechanism (new Price on same Product) is Stripe's documented non-breaking pattern per BLDR-08. No automatic migration in v1 — deferred. |
| T-03-04-06 | Elevation of Privilege | Un-onboarded clinic calls updatePlanPrices | mitigate | Same isPublishReady() gate as publishPlan. If clinic loses Connect capabilities after publish, price edits are blocked. |
| T-03-04-07 | Information Disclosure | Old price history JSON leaks via API response | mitigate | Server action returns only { ok: true, updatedTiers: [{ tierId, newPriceId, newUnitAmountCents }] } — never the full history; never other tiers in other clinics. |
| T-03-04-08 | Tampering | Rename-tier action renames after publish — corrupts Stripe Product name drift | mitigate | renameTiers is gated on plan.status === 'draft'. Published plans refuse name edits. (BLDR-06 scopes to pre-publish explicitly.) |
</threat_model>

```typescript
/**
 * Phase 3 BLDR-08: derive fee + gross + break-even cents for a SINGLE tier
 * given a new monthly-fee amount (cents). Used by updatePlanPrices to keep
 * PlanTier derived columns in sync with Stripe's new Price amount.
 *
 * Mirrors the per-tier arithmetic inside computeTier() but skips the
 * retail-value + service-list derivation (those stay unchanged on a
 * price-only edit — only the monthly fee moves).
 */
export interface TierDerivedCents {
  stripeFeePerChargeCents: number;
  platformFeePerChargeCents: number;
  clinicGrossPerPetPerYearCents: number;
  breakEvenMembers: number;
}

export function deriveTierFromMonthlyFeeCents(args: {
  monthlyFeeCents: number;
  monthlyProgramOverheadUsd: number;
}): TierDerivedCents {
  const monthlyFeeUsd = args.monthlyFeeCents / 100;
  const stripeFeeUsd = Math.round((monthlyFeeUsd * (STRIPE_FEE_PCT / 100) + STRIPE_FEE_FIXED_CENTS / 100) * 100) / 100;
  const platformFeeUsd = Math.round(monthlyFeeUsd * (PLATFORM_FEE_PCT / 100) * 100) / 100;
  const grossUsd = Math.round((monthlyFeeUsd - stripeFeeUsd - platformFeeUsd) * 12 * 100) / 100;
  const breakEvenMembers =
    args.monthlyProgramOverheadUsd === 0
      ? 0
      : grossUsd > 0
        ? Math.ceil((args.monthlyProgramOverheadUsd * 12) / grossUsd)
        : Number.POSITIVE_INFINITY;
  return {
    stripeFeePerChargeCents: Math.round(stripeFeeUsd * 100),
    platformFeePerChargeCents: Math.round(platformFeeUsd * 100),
    clinicGrossPerPetPerYearCents: Math.round(grossUsd * 100),
    breakEvenMembers,
  };
}
```

```typescript
// ... existing imports ...
import { deriveTierFromMonthlyFeeCents } from '@/lib/pricing/breakEven';

const RenameTiersSchema = z.object({
  planId: z.string().uuid(),
  renames: z
    .array(
      z.object({
        tierId: z.string().uuid(),
        tierName: z.string().trim().min(1).max(40).regex(/^[^<>]+$/, 'No < or > in tier names'),
      }),
    )
    .min(1)
    .max(3),
});

const UpdatePricesSchema = z.object({
  planId: z.string().uuid(),
  priceChanges: z
    .array(
      z.object({
        tierId: z.string().uuid(),
        newMonthlyFeeCents: z.number().int().min(500).max(100_000),
      }),
    )
    .min(1)
    .max(3),
});

export async function renameTiers(input: z.infer<typeof RenameTiersSchema>): Promise<
  | { ok: true }
  | { ok: false; code: PublishErrorCode; error: string }
> {
  const parsed = RenameTiersSchema.safeParse(input);
  if (!parsed.success) return { ok: false, code: 'VALIDATION_FAILED', error: parsed.error.issues[0]?.message ?? 'Invalid input' };

  let clinic;
  try { clinic = await requireClinic(); }
  catch (e) {
    const msg = (e as Error).message;
    if (msg === 'UNAUTHENTICATED') return { ok: false, code: 'UNAUTHENTICATED', error: 'Please log in.' };
    if (msg === 'NO_CLINIC') return { ok: false, code: 'NO_CLINIC', error: 'No clinic found.' };
    throw e;
  }

  try {
    await withClinic(clinic.id, async (tx) => {
      const plan = await tx.plan.findFirst({
        where: { id: parsed.data.planId, clinicId: clinic.id },
        include: { tiers: true },
      });
      if (!plan) throw new Error('NO_DRAFT_PLAN');
      if (plan.status !== 'draft') throw new Error('ALREADY_PUBLISHED');

      for (const r of parsed.data.renames) {
        const belongs = plan.tiers.some((t) => t.id === r.tierId);
        if (!belongs) throw new Error('NO_DRAFT_PLAN');
        await tx.planTier.update({ where: { id: r.tierId }, data: { tierName: r.tierName } });
      }
    });
    return { ok: true };
  } catch (e) {
    const msg = (e as Error).message;
    if (msg === 'NO_DRAFT_PLAN') return { ok: false, code: 'NO_DRAFT_PLAN', error: 'Draft not found.' };
    if (msg === 'ALREADY_PUBLISHED') return { ok: false, code: 'ALREADY_PUBLISHED', error: 'Plan already published. Tier names are locked.' };
    console.error('[renameTiers] failed', { clinicId: clinic.id, err: e });
    return { ok: false, code: 'VALIDATION_FAILED', error: 'Something went wrong. Please try again.' };
  }
}

export async function updatePlanPrices(
  input: z.infer<typeof UpdatePricesSchema>,
): Promise<
  | { ok: true; updatedTiers: Array<{ tierId: string; newPriceId: string; newUnitAmountCents: number }> }
  | { ok: false; code: PublishErrorCode; error: string }
> {
  const parsed = UpdatePricesSchema.safeParse(input);
  if (!parsed.success) return { ok: false, code: 'VALIDATION_FAILED', error: parsed.error.issues[0]?.message ?? 'Invalid input' };

  let clinic;
  try { clinic = await requireClinic(); }
  catch (e) {
    const msg = (e as Error).message;
    if (msg === 'UNAUTHENTICATED') return { ok: false, code: 'UNAUTHENTICATED', error: 'Please log in.' };
    if (msg === 'NO_CLINIC') return { ok: false, code: 'NO_CLINIC', error: 'No clinic found.' };
    throw e;
  }

  if (
    !isPublishReady({
      chargesEnabled: clinic.stripeChargesEnabled,
      payoutsEnabled: clinic.stripePayoutsEnabled,
      disabledReason: clinic.stripeDisabledReason,
    })
  ) {
    return { ok: false, code: 'NOT_PUBLISH_READY', error: "Your Stripe account isn't ready." };
  }

  // Load the plan + tiers (under RLS).
  const plan = await withClinic(clinic.id, (tx) =>
    tx.plan.findFirst({
      where: { id: parsed.data.planId, clinicId: clinic.id, status: 'published' },
      include: { tiers: true },
    }),
  );
  if (!plan) return { ok: false, code: 'NO_DRAFT_PLAN', error: 'Published plan not found.' };

  const overheadUsd = Number(plan.monthlyProgramOverheadUsd);

  // Filter to actual changes.
  type TierRow = (typeof plan.tiers)[number];
  const changes: Array<{ tier: TierRow; newCents: number; nextVersion: number }> = [];
  for (const c of parsed.data.priceChanges) {
    const tier = plan.tiers.find((t) => t.id === c.tierId);
    if (!tier) return { ok: false, code: 'NO_DRAFT_PLAN', error: 'Tier not found in this plan.' };
    if (!tier.stripeProductId || !tier.stripePriceId) {
      return { ok: false, code: 'VALIDATION_FAILED', error: 'Tier is missing Stripe Product. Re-publish first.' };
    }
    if (tier.monthlyFeeCents === c.newMonthlyFeeCents) continue;
    const history = (tier.stripePriceHistory as PublishedPriceHistoryEntry[] | null) ?? [];
    changes.push({ tier, newCents: c.newMonthlyFeeCents, nextVersion: history.length + 1 });
  }
  if (changes.length === 0) {
    return { ok: true, updatedTiers: [] };
  }

  // Create new Prices on the same Products (platform account, idempotent).
  const stripeResults: Array<{ tierId: string; newPriceId: string; newUnitAmountCents: number; createdAtIso: string; nextVersion: number }> = [];
  for (const ch of changes) {
    try {
      const price = await createPlatformPrice({
        productId: ch.tier.stripeProductId!,
        unitAmountCents: ch.newCents,
        metadata: {
          clinicId: clinic.id,
          planId: plan.id,
          tierId: ch.tier.id,
          version: ch.nextVersion,
        },
        idempotencyKey: `price-edit:${plan.id}:${ch.tier.id}:v${ch.nextVersion}:${ch.newCents}`,
      });
      stripeResults.push({
        tierId: ch.tier.id,
        newPriceId: price.id,
        newUnitAmountCents: ch.newCents,
        createdAtIso: new Date(price.created * 1000).toISOString(),
        nextVersion: ch.nextVersion,
      });
    } catch (e) {
      console.error('[updatePlanPrices] prices.create failed', { clinicId: clinic.id, tierId: ch.tier.id, err: e });
      return { ok: false, code: 'STRIPE_PRICE_CREATE_FAILED', error: 'Price update failed. Please try again.' };
    }
  }

  // Atomic DB commit.
  try {
    await withClinic(clinic.id, async (tx) => {
      for (const r of stripeResults) {
        const tier = changes.find((c) => c.tier.id === r.tierId)!.tier;
        const oldHistory = (tier.stripePriceHistory as PublishedPriceHistoryEntry[] | null) ?? [];
        const updatedOldHistory = oldHistory.map((e, i) =>
          i === oldHistory.length - 1 && e.replacedAt === null ? { ...e, replacedAt: new Date().toISOString() } : e,
        );
        const newHistory: PublishedPriceHistoryEntry[] = [
          ...updatedOldHistory,
          { priceId: r.newPriceId, unitAmountCents: r.newUnitAmountCents, createdAt: r.createdAtIso, replacedAt: null },
        ];
        const derived = deriveTierFromMonthlyFeeCents({ monthlyFeeCents: r.newUnitAmountCents, monthlyProgramOverheadUsd: overheadUsd });
        await tx.planTier.update({
          where: { id: r.tierId },
          data: {
            stripePriceId: r.newPriceId,
            stripePriceHistory: newHistory as unknown as Prisma.InputJsonValue,
            monthlyFeeCents: r.newUnitAmountCents,
            stripeFeePerChargeCents: derived.stripeFeePerChargeCents,
            platformFeePerChargeCents: derived.platformFeePerChargeCents,
            clinicGrossPerPetPerYearCents: derived.clinicGrossPerPetPerYearCents,
            breakEvenMembers: derived.breakEvenMembers === Number.POSITIVE_INFINITY ? 0 : derived.breakEvenMembers,
          },
        });
      }
    });
  } catch (e) {
    console.error('[updatePlanPrices] DB commit failed AFTER Stripe create', {
      clinicId: clinic.id,
      planId: plan.id,
      orphaned: stripeResults.map((r) => ({ priceId: r.newPriceId })),
      err: e,
    });
    return { ok: false, code: 'VALIDATION_FAILED', error: 'Price update failed. Please try again.' };
  }

  revalidateTag(`clinic:${clinic.slug}`);

  return {
    ok: true,
    updatedTiers: stripeResults.map((r) => ({ tierId: r.tierId, newPriceId: r.newPriceId, newUnitAmountCents: r.newUnitAmountCents })),
  };
}
```

```typescript
import type { ReactNode } from 'react';

interface Props {
  retailValueBundledCents: number;
  monthlyFeeCents: number;
  stripeFeePerChargeCents: number;
  platformFeePerChargeCents: number;
  clinicGrossPerPetPerYearCents: number;
  breakEvenMembers: number;
  accentHex?: string;
}

const usd = (cents: number) => `$${(cents / 100).toFixed(2)}`;

function Row({ label, value, negative = false, emphasis = false, accentHex }: { label: string; value: ReactNode; negative?: boolean; emphasis?: boolean; accentHex?: string }) {
  return (
    <div className="flex items-center justify-between py-2 text-sm">
      <span className="text-[#1C1B18]">{label}</span>
      <span
        className={[
          'font-mono tabular-nums',
          emphasis ? 'text-xl font-semibold' : '',
          negative ? 'text-[#6B6A63]' : '',
        ].join(' ')}
        style={emphasis && accentHex ? { color: accentHex } : undefined}
      >
        {value}
      </span>
    </div>
  );
}

export function BreakEvenLineItems(props: Props) {
  const {
    retailValueBundledCents,
    monthlyFeeCents,
    stripeFeePerChargeCents,
    platformFeePerChargeCents,
    clinicGrossPerPetPerYearCents,
    breakEvenMembers,
    accentHex = '#2F7D6E',
  } = props;

  const isImpossible = breakEvenMembers === 0 || breakEvenMembers > 99_999;

  return (
    <div className="divide-y divide-[#E8E6E0]">
      <Row label="Retail value bundled" value={`${usd(retailValueBundledCents)} / yr`} />
      <Row label="Monthly fee (member pays)" value={`${usd(monthlyFeeCents)} / mo`} />
      <Row label="Stripe processing estimate" value={`−${usd(stripeFeePerChargeCents)} / charge`} negative />
      <Row label="PawPlan platform fee (10%)" value={`−${usd(platformFeePerChargeCents)} / charge`} negative />
      <Row label="Clinic gross per pet per year" value={`${usd(clinicGrossPerPetPerYearCents)} / yr`} />
      <Row
        label="Members to break even"
        value={isImpossible ? 'Monthly fee too low' : `${breakEvenMembers}`}
        emphasis
        accentHex={accentHex}
      />
    </div>
  );
}
```

**3. Create `src/app/(dashboard)/dashboard/plans/_components/EditTierPricesDialog.tsx`:**

```typescript
'use client';

import { useState, useTransition } from 'react';
import { toast } from 'sonner';
import { Dialog, DialogContent, DialogHeader, DialogTitle, DialogDescription, DialogFooter } from '@/components/ui/dialog';
import { Button } from '@/components/ui/button';
import { Input } from '@/components/ui/input';
import { Label } from '@/components/ui/label';
import { updatePlanPrices } from '@/app/actions/publish';

interface TierOption {
  tierId: string;
  tierName: string;
  currentMonthlyFeeCents: number;
}

interface Props {
  open: boolean;
  onOpenChange: (open: boolean) => void;
  planId: string;
  tiers: TierOption[];
}

const centsToDollarString = (c: number) => (c / 100).toFixed(2);
const dollarStringToCents = (s: string): number | null => {
  const n = Number(s);
  if (!Number.isFinite(n)) return null;
  return Math.round(n * 100);
};

export function EditTierPricesDialog({ open, onOpenChange, planId, tiers }: Props) {
  const [values, setValues] = useState<Record<string, string>>(() =>
    Object.fromEntries(tiers.map((t) => [t.tierId, centsToDollarString(t.currentMonthlyFeeCents)])),
  );
  const [pending, startTransition] = useTransition();

  const onSubmit = () => {
    const changes = tiers.flatMap((t) => {
      const cents = dollarStringToCents(values[t.tierId] ?? '');
      if (cents === null) return [];
      if (cents === t.currentMonthlyFeeCents) return [];
      return [{ tierId: t.tierId, newMonthlyFeeCents: cents }];
    });
    if (changes.length === 0) {
      toast.info('No changes to save.');
      onOpenChange(false);
      return;
    }
    startTransition(async () => {
      const res = await updatePlanPrices({ planId, priceChanges: changes });
      if (res.ok) {
        toast.success(`Updated ${res.updatedTiers.length} tier price(s).`);
        onOpenChange(false);
      } else {
        toast.error(res.error);
      }
    });
  };

  return (
    <Dialog open={open} onOpenChange={onOpenChange}>
      <DialogContent>
        <DialogHeader>
          <DialogTitle>Edit plan prices</DialogTitle>
          <DialogDescription>
            <strong>Applies to new enrollments only.</strong> Existing members keep their current price until they cancel and re-enroll.
          </DialogDescription>
        </DialogHeader>
        <div className="space-y-4 py-2">
          {tiers.map((t) => (
            <div key={t.tierId} className="grid grid-cols-[1fr_120px] items-center gap-3">
              <Label htmlFor={`price-${t.tierId}`}>{t.tierName}</Label>
              <div className="relative">
                <span className="pointer-events-none absolute left-3 top-1/2 -translate-y-1/2 text-[#6B6A63]">$</span>
                <Input
                  id={`price-${t.tierId}`}
                  type="number"
                  step="0.01"
                  min="5.00"
                  max="1000.00"
                  inputMode="decimal"
                  className="pl-7 font-mono"
                  value={values[t.tierId] ?? ''}
                  onChange={(e) => setValues((v) => ({ ...v, [t.tierId]: e.target.value }))}
                />
              </div>
            </div>
          ))}
        </div>
        <DialogFooter>
          <Button variant="outline" onClick={() => onOpenChange(false)} disabled={pending}>Cancel</Button>
          <Button onClick={onSubmit} disabled={pending}>{pending ? 'Saving…' : 'Save new prices'}</Button>
        </DialogFooter>
      </DialogContent>
    </Dialog>
  );
}
```

import { Button } from '@/components/ui/button';

    </section>
  );
